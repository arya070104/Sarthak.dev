# sarthak.dev

A responsive personal portfolio built with semantic HTML, modern CSS, and lightweight JavaScript. Content is based on Sarthak's latest résumé.

Live site: [arya070104.github.io/Sarthak.dev](https://arya070104.github.io/Sarthak.dev/)

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
